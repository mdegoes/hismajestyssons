Dev Tool — not part of the system

05 / Reference only

tweaks-panel.jsx ships its own complete, self-contained control kit (slider, segmented radio, toggle, color chips, number stepper, text field, buttons) used by the floating panel on the home page. It's deliberately unthemed — its colors, blur, and shadow values are hardcoded rather than drawn from --paper/--ink/--accent, so it looks the same regardless of site scheme. Treat it as a portable dev tool, not a component to theme-match or extend for public pages.
